Real Work

Real work is a full-stack web application that help users track work/study sessions and enhance productivity. Unlike traditional time tracking apps, Real Work introduces novel features such as intensity tracking, enabling users to quantify work/study quality. Additionally, Real Work provides flexibility in time management, allowing users to freely customize their work/study sessions, different from traditional pomodoro timers with fixed time intervals.


Tech Stack: HTML, CSS, JavaScript
